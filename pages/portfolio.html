CMS API

The goal of this project is to create a self-hosted CMS solution for freelance web developers for managing content for multiple clients. This will also include an admin interface application and will emphasize the ability to allow clients themselves to log into the admin interface and create new entries for content types they have permissions for. Additionally, the goal of the project is to support different databases, be easily modifiable, scalable, and open source to the public.
