    Tetris 2P AI

    After the loss of tetrisfriends.com, I wanted to develop my own Tetris client with the Tetris Battle 2P format/rule set along with a computer AI built from scratch. Following Google’s AlphaStar as an example, the end goal is to teach the AI different Tetris strategies, from simply surviving to 4-wide and T-Spin strategies. I also hope to exercise my AI knowledge from college courses as well as to learn more about game development architectures.

    CMS API

    The goal of this project is to create a self-hosted CMS solution for freelance web developers for managing content for multiple clients. This will also include an admin interface application for creating new content types and adding more entries. The project will emphasize the ability to allow clients themselves to log into the admin interface and create new entries for content types they have permissions for. Another goal of the project is to support different databases and be easily modifiable for other developers.
