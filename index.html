2Day's New Tab Page


2Day's New Tab replaces the default chrome new tab page. Personalize your own new tab page with a custom wallpaper you get to upload. 2Day displays your Google Calendar events in a fun responsive schedule interface. It also displays your important bookmarks as well as bookmark folders. Clicking on a bookmark folder will open each bookmark inside in a new tab. A current and weekly weather forecast is displayed in the top right for any user specified location. Lastly, time and date is displayed in a large clock in the bottom left corner.
